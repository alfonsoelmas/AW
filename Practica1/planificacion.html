Planificacion del proyecto


La planificacion del proyecto la dividimos en varias partes.

Tareas a realizar

Las tareas a realizar seran crear diferentes modulos que gestionen dinamicamente partes
de nuestra pagina web (PHP), asi como generar diferentes estilos(CSS) y una estructura general
para mantener un disenio coherente. Es posible que hagamos uso de boostrap para mantener dicho

asi como animaciones, haremos uso de JS, seguramente implementaremos JQuery por ser una libreria
muy potente para este lenguaje.

Como repartir el trabajo

En general, se realizara un esquema principal con todos los archivos de codigo que haya
que ir creando (En html, php, JS, CSS, etc) con sus respectivas funciones y funcionalidades,
realizando un proyecto modularizado. Tras este esquema nos dividiremos equitativamente el
trabajo tocando cada miembro del grupo algo de cada lenguaje, para asi ampliar conocimientos
generales. Cada miembro del grupo podra tambien hacer revisiones sobre los demas archivos y
modificarlos para su mejora, mediante la plataforma de github.

Plazos del proyecto

Como el proyecto se va a estructurar a medida que adquiramos los conocimientos competentes de
cada lenguaje, iremos realizando el proyecto en diferentes plazos.

El primer plazo sera tener la estructura de la web creada en HTML y a medida que este finalizando
comenzara un segundo plazo que sera ir dandole estilo y aplicar CSS sobre la web.

Lo siguiente sera dinamizarla mediante uso de PHP, esto requerira modificar pequenios fragmentos de
nuestro HTML y nuestro codigo.

Finalmente, se introducira JS en nuestra aplicacion web para darle aun mayor interactividad con el
usuario, asi como hacer revisiones y modificaciones en el codigo ya creado para mejorar y corregir la web.
