Planificacion del proyecto


La planificación del proyecto la dividimos en varias partes.

Tareas a realizar

Las tareas a realizar seran crear diferentes modulos que gestionen dinamicamente partes
de nuestra pagina web (PHP), asi como generar diferentes estilos(CSS) y una estructura general
para mantener un diseño coherente. Es posible que hagamos uso de boostrap para mantener dicho
diseño (Gracias a su tematica responsive). Finalemente, para darle mayor interactividad al usuario
asi como animaciones, haremos uso de JS, seguramente implementaremos JQuery por ser una libreria
muy potente para este lenguaje.

Como repartir el trabajo

En general, se realizará un esquema principal con todos los archivos de código que haya
que ir creando (En html, php, JS, CSS…) con sus respectivas funciones y funcionalidades,
realizando un proyecto modularizado. Tras este esquema nos dividiremos equitativamente el
trabajo tocando cada miembro del grupo algo de cada lenguaje, para así ampliar conocimientos
generales. Cada miembro del grupo podrá también hacer revisiones sobre los demás archivos y
modificarlos para su mejora, mediante la plataforma de github.
